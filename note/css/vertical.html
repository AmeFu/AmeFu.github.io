# 水平居中对齐

文本居中对齐，可以使用text-align【行内元素及行内块元素居中对齐和文本相同】

div {
    text-align: center;
}

# 垂直居中对齐

高度固定时，可以设置间距来实现，高度不固定时，参考水平垂直居中使用position居中定位后偏移自身一半来对齐。

div.parent {
    box-sizing: border-box;
    height: 100px;
    padding-top: 25px;
}
div.child {
    height: 50px;
    width: 50px;
}
